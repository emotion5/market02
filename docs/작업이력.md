## 2026-07-16

**타이포그래피 전면 정리 — 얇고 고급스러운 톤**
- 본문 서체를 **Pretendard**로 교체 (자체 호스팅). 한글 서체 특성상 글자 범위별로 쪼개 두어, 페이지마다 **필요한 부분만 내려받는다**
- 사이트 전반의 **글자 굵기를 한 단계씩 낮춤** — 상품명·라벨·내비게이션은 보통 굵기로, 가격·소제목은 중간 굵기로
- 대신 최상단 대형 **Products** 타이틀만 굵게 남겨, "큰 제목 하나만 세우고 나머지는 평평하게" 라는 레퍼런스의 위계를 적용
- 앞으로 굵기 조절은 **한 곳(`globals.css`)의 값 세 개**로 사이트 전체에 반영됨

**메인 카테고리 표기**
- 메인페이지 카테고리 제목에 **한글 아래 영문명 병기** (예: 하드웨어 / Hardware) — 상단 내비와 같은 영문명 사용

**상품 목록 정리**
- 상품 카드·목록에서 **[오늘출발] 평일 14시 마감 표기 삭제** (카드형·목록형 모두)
- 상세페이지 '배송' 안내의 당일출발 설명은 그대로 유지

**견적서를 문서로 승격 — 작성 중 / 발행 이력 분리**
- 견적서 하단에 **'견적서 발행' 버튼** 추가 — 누르면 그 시점 내용 그대로 **견적번호·발행일이 확정**되어 저장됨
- 마이페이지에 **'견적서 내역' 메뉴** 신설 — 발행한 견적서를 최신순으로 보관, 다시 열어 인쇄하거나 주문으로 전환 가능
- 발행본은 **수정 불가(읽기 전용)**. 발행 당시의 가격·품목이 그대로 보존되어, 고객에게 보낸 문서와 나중에 열어본 문서가 항상 같음
- 유효기간이 지난 견적서는 **'유효기간 만료'로 표시**
- ※ 이전에는 견적서를 열 때마다 견적번호와 발행일이 새로 생성되어, 고객에게 보낸 견적서를 다시 찾거나 대조할 수 없었음 — 이번 작업으로 해결

**버그 수정**
- 마이페이지 결제내역·견적서 내역 목록 왼쪽에 **불필요한 목록 점(•)이 보이던 문제 수정**
- 헤더의 **로고·검색·견적서·계정·로그인이 클릭되지 않던 문제 수정** — 화면 상단에 고정된
  카테고리 알약의 투명 영역이 화면 전폭을 덮어 같은 높이의 로고·아이콘 클릭을 가로채고 있었음
  (헤더 바를 없애고 로고·아이콘을 셸 격자에 직접 배치한 이후 발생)
- 마이페이지 **결제내역의 일부 상품 썸네일이 깨지던 문제 수정** — 개발용 시드 데이터가 이미지
  경로를 복제해두어 상품 이미지 파일이 바뀐 뒤 옛 경로에 남아 있었음. 시드가 상품 데이터에서
  이미지를 직접 참조하도록 바꿔 같은 문제가 재발하지 않게 함
  - ※ 이미 주입해 둔 목업 데이터에는 옛 경로가 남아 있으므로 `/dev/seed`에서 **다시 시드**해야 반영됨

## 2026-07-15

**아이콘 · 톤 정리**
- 헤더(검색·견적서·계정·로그인)와 상세설명 아코디언의 **+/− 아이콘을 얇게** 조정 — 선 굵기를 낮춰 전체적으로 가벼운 톤으로 통일
- 상품 카드의 **견적서 바로 추가 아이콘 축소**(34→27px), 아이콘 자체도 작게

**상품 상세**
- 수량을 **직접 입력** 가능하도록 변경 — 테두리 없이 얇은 +/− 아이콘과 숫자만 노출(아코디언과 같은 톤)
- 상세 모달에서 첫 **색상 스와치의 선택 링이 잘리던 문제** 수정

**견적서 패널**
- 패널 외곽을 **위에서 아래로 옅어지는 그라데이션 1px 링**으로 변경
- 헤더를 검정 배경에서 **밝은 회색 면**으로 변경 — 목록과 톤 차이만으로 구분
- 내부 구분선을 **양 끝으로 사라지는 헤어라인**으로 변경, 선 색도 한 단계 옅게
- 모바일 바텀시트에서는 화면 폭을 꽉 채우므로 **외곽 링을 숨김**(내부 구분선은 유지)

**상품 목록 · 카드**
- 레퍼런스처럼 **상품 그리드 배경만 흰색 판**으로 분리 — 카테고리명 열은 회색 배경 유지
- 카드 마우스오버 **그림자를 25% 축소하고 농도도 옅게** 조정
- 카드 **행 간격을 좁힘**(32→26px). 열 간격은 마우스오버 패널이 옆 카드를 침범하지 않는 하한선이라 20px 유지

**상품 · 이미지**
- 수전 카테고리에 **상품 4종 추가** — 새 사진 4장이 목록 앞쪽에 노출
- 수전 대표 이미지를 **전부 흰 배경으로 교체** — 배경이 투명하거나 회색이던 사진 정리

**브랜드**
- 헤더 로고를 **가로형 로고 이미지**로 교체, 기존 심볼+워드마크 조합 제거

**헤더 · 내비게이션 개편**
- 카테고리 내비 바를 **반투명 유리(프로스티드 글라스)** 로 변경 — 뒤로 지나가는 상품이 흐릿하게 비침
- **헤더 바를 없앰** — 흰 배경과 경계선을 걷어내 헤더·내비·상품 리스트가 한 화면으로 이어짐
- 로고를 **왼쪽 견적서 열 위**로 이동, 아이콘은 콘텐츠 우측 상단에 배치 (레퍼런스 구조)
- 왼쪽 견적서 열은 **흰색**, 오른쪽은 **회색**이 화면 맨 위부터 아래까지 이어지는 2단 배경
- 헤더가 **스크롤과 함께 사라지고**, 상단에는 카테고리 내비만 남음

## 2026-07-14

**상품 상세 · 이미지**
- 상품 상세를 **모달(팝업형 오버레이)** 로 열도록 개선 — 목록 위에 겹쳐 뜨고 배경이 어두워짐. 직접 URL 접근·새로고침 시엔 일반 상세 페이지로 표시(공유·SEO 유지)
- 상세 모달 톤을 **밝은 회색**으로, **X 닫기 버튼**을 우측 상단 코너 원형 버튼으로 조정
- 모달에서 내용이 길어지면(아코디언 펼침 등) **오른쪽 정보 영역만 내부 스크롤** — 전체 페이지·모달이 늘어나지 않음
- 상품 상세 오른쪽에 **배송·상세설명·크기 아코디언**(+/− 펼침) 추가
- 상품 상세 **다중 이미지 갤러리** 추가 — 메인 이미지 아래 썸네일 1행(최대 6장), 클릭 시 전환. 파일명 규칙(`상품id-1`, `-2`…)으로 자동 인식
- `faucet-0001` 상품에 상세 이미지 5장 추가
- 옵션이 하나뿐인 **단독 상품**은 옵션 선택 없이 소비자가·회원도매가·수량만 바로 표시

**상품 목록 · 카드**
- 카드 썸네일 우측 상단에 **견적서 바로 추가 아이콘**(+툴팁) 추가 — 상세에 안 들어가도 바로 담김
- 카드 **테두리 제거**, 마우스오버 시 **그림자로 떠오르는** 효과로 변경. 추가 아이콘도 그림자 스타일로 통일
- 상품 그리드 **열 수 자동 조정** + 페이지 최대폭 확대(1200→1600px) — 맥북에어에서 4열 노출
- 홈 카테고리 섹션을 좌측 카테고리명 + 우측 그리드 **2단 레이아웃**으로 변경

**내비게이션 · 레이아웃**
- 상단 카테고리 내비 개편 — '전체' 칩 제거, **흰색 알약(pill)형 바**, 마우스오버 시 **영문↔한글 전환**
- 오른쪽 콘텐츠 영역 **흐린 회색 배경**, 메인 최상단 대형 **Products** 타이틀 추가
- 브랜드 심볼을 **파비콘**(주소창 아이콘)으로 적용

**회원 · 로그인**
- 회원가입을 **개인/사업자 유형 선택** 구조로 분리 (`/signup` → `/signup/personal`, `/signup/business`), 헤더 버튼 '회원가입'으로 변경
- **이메일을 공용 계정 아이디(primary key)** 로 정리 — 로그인은 이메일 단일 입력, 사업자 가입에도 이메일 추가(사업자등록번호는 확인용)
- 개인회원 가입은 이메일·비밀번호만 (이름은 주문 단계에서 수집)
