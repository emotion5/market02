# MMM MARKET — 사이트맵

전체 페이지(라우트) 구조입니다. 경로는 Next.js App Router 기준입니다.

> 표기: `[slug]`, `[id]` 는 동적 경로(값에 따라 페이지 생성),
> `(with-quote)` 는 URL에 노출되지 않는 레이아웃 그룹(좌측 견적 패널 공유).

---

## 전체 구조

```
/                                  홈 — 전체 상품 목록 (좌측 견적 패널)
│
├─ /products                       전체 상품
│   └─ /products/[id]              상품 상세  (동적)
├─ /category/[slug]                카테고리별 상품  (동적)
│     ├─ /category/faucet          · 수전
│     ├─ /category/flooring        · 바닥재
│     ├─ /category/gangmaru        · 강마루
│     ├─ /category/jangpan         · 장판
│     ├─ /category/hardware        · 하드웨어
│     └─ /category/drawer          · 서랍
├─ /search?q=…                     검색 결과
│
├─ /quote                          견적서 (A4, 수량 편집 · 인쇄 · PDF)
├─ /checkout                       주문 / 결제 (무통장입금)
│
├─ 계정
│   ├─ /login                      로그인 (아이디 = 사업자등록번호)
│   ├─ /business-signup            기업회원 가입
│   └─ /find-account               아이디·비밀번호 찾기 (문의 안내)
│
├─ /mypage                         → /mypage/orders 로 이동
│   ├─ /mypage/orders              결제내역 · 배송조회
│   ├─ /mypage/tax-invoices        세금계산서 내역
│   ├─ /mypage/addresses           배송지 관리
│   ├─ /mypage/profile             회원정보 수정
│   └─ /mypage/withdraw            회원 탈퇴
│
└─ /dev/seed                       [개발용] 목업 데이터 시드
```

## 페이지별 설명

### 상품 · 탐색

| 경로 | 설명 | 렌더링 |
|------|------|--------|
| `/` | 홈. 전체 상품 목록 (카드/리스트 전환) | 정적 |
| `/products` | 전체 상품 목록 | 정적 |
| `/products/[id]` | 상품 상세 — 옵션·색상 선택 후 견적 담기 | 동적 |
| `/category/[slug]` | 카테고리별 목록 (6개 카테고리) | 정적 생성(SSG) |
| `/search` | 검색 결과 (`?q=` 쿼리) | 동적 |

### 견적 · 주문

| 경로 | 설명 |
|------|------|
| `/quote` | A4 종이 견적서. 화면에서 수량 편집·삭제, 인쇄 / PDF 저장, 하단에 주문 버튼 |
| `/checkout` | 주문/결제. 배송정보·무통장입금·세금계산서 → 주문 완료 |

### 계정

| 경로 | 설명 |
|------|------|
| `/login` | 로그인 (사업자등록번호 + 비밀번호) |
| `/business-signup` | 기업회원 가입 (사업자번호·비밀번호·사업자등록증, 영업일 1일 승인) |
| `/find-account` | 아이디·비밀번호 분실 시 고객센터 문의 안내 |

### 마이페이지

| 경로 | 설명 |
|------|------|
| `/mypage` | 진입 시 결제내역으로 리다이렉트 |
| `/mypage/orders` | 결제내역 (견적 단위) + 배송상태 뱃지 + 배송조회 타임라인 |
| `/mypage/tax-invoices` | 세금계산서 발행 내역 |
| `/mypage/addresses` | 배송지 등록·삭제·기본배송지 설정 |
| `/mypage/profile` | 상호·담당자·연락처·이메일·비밀번호 변경 (아이디는 변경 불가) |
| `/mypage/withdraw` | 회원 탈퇴 |

### 기타

| 경로 | 설명 |
|------|------|
| `/dev/seed` | **개발용.** `data/seed/*.json` 을 localStorage에 주입해 목업 사용자·주문·배송지 테스트. 배포 시 제외 대상 |

---

## 진입 경로 (헤더/화면 내 링크)

- **헤더 우측:** 검색 · 견적서(`/quote`) · 마이페이지 · 로그인 · **기업회원 가입**
- **좌측 견적 패널:** "견적서 보기" → `/quote`
- **`/quote`:** 수량 편집·삭제, "주문 / 결제하기" → `/checkout`
- **`/checkout` 완료:** "주문내역 보기" → `/mypage/orders`
