# MMM MARKET — 사이트맵

전체 페이지(라우트) 구조입니다. 경로는 Next.js App Router 기준입니다.

> 표기: `[id]`, `[slug]`, `[number]` 는 동적 경로(값에 따라 페이지 생성).
> `(with-quote)`·`(plain)` 은 URL에 노출되지 않는 레이아웃 그룹.
> `(with-quote)` = 좌측 견적 패널을 공유하는 탐색 화면, `(plain)` = 그 외 단독 화면.

---

## 전체 구조

```
/                                  홈 — 카테고리별 큐레이션 섹션 (좌측 견적 패널)     (with-quote)
│
├─ 탐색                                                                        (with-quote)
│   ├─ /products                    전체 상품
│   ├─ /products/[id]               상품 상세  (동적)
│   │      └─ (목록에서 클릭)        상품 상세 모달 — 인터셉트 라우트
│   ├─ /category/[slug]             카테고리별 상품  (동적 · SSG)
│   │      ├─ /category/hardware    · 하드웨어
│   │      ├─ /category/panel       · 가구재
│   │      ├─ /category/pressbevel  · 바닥재
│   │      ├─ /category/antipress   · 장판
│   │      ├─ /category/bathmatch   · 위생
│   │      └─ /category/stablecore  · 건자재
│   └─ /search?q=…                  검색 결과
│
├─ 견적 · 주문                                                                 (plain)
│   ├─ /quote                       현재 견적 (수량 편집 · 인쇄 · PDF · 발행)
│   ├─ /quotes/[number]             발행된 견적서 열람 (읽기 전용, 동적)
│   └─ /checkout                    주문 / 결제 (무통장입금)
│
├─ 계정                                                                        (plain)
│   ├─ /login                       로그인 (이메일 + 비밀번호)
│   ├─ /signup                      회원가입 — 유형 선택
│   │      ├─ /signup/personal      · 개인회원 (이메일로 즉시 가입)
│   │      └─ /signup/business      · 사업자회원 (승인 후 회원도매가)
│   └─ /find-account                아이디·비밀번호 찾기 (문의 안내)
│
├─ /mypage                          → /mypage/orders 로 이동                    (plain)
│   ├─ /mypage/quotes               견적서 내역
│   ├─ /mypage/orders               결제내역 · 배송조회
│   ├─ /mypage/tax-invoices         세금계산서 내역
│   ├─ /mypage/addresses            배송지 관리
│   ├─ /mypage/profile              회원정보 수정
│   └─ /mypage/withdraw             회원 탈퇴
│
└─ /dev/seed                        [개발용] 목업 데이터 시드                    (plain)
```

## 페이지별 설명

### 상품 · 탐색  `(with-quote)`

| 경로 | 설명 | 렌더링 |
|------|------|--------|
| `/` | 홈. 카테고리별 큐레이션 섹션 (`data/featured.json` 순서) | 정적 |
| `/products` | 전체 상품 목록 | 정적 |
| `/products/[id]` | 상품 상세 — 옵션·색상 선택 후 견적 담기 | 동적 |
| `/products/[id]` (모달) | 목록에서 상품 클릭 시 인터셉트 라우트로 상세를 모달 표시. 새로고침·직접 접근 시 실제 상세 페이지로 열림 | 동적 |
| `/category/[slug]` | 카테고리별 목록 (6개 카테고리) | 정적 생성(SSG) |
| `/search` | 검색 결과 (`?q=` 쿼리) | 동적 |

> 카테고리 slug ↔ 표시 이름은 `lib/constants.ts` 에서 관리합니다.
> `hardware`=하드웨어, `panel`=가구재, `pressbevel`=바닥재, `antipress`=장판, `bathmatch`=위생, `stablecore`=건자재.

### 견적 · 주문  `(plain)`

| 경로 | 설명 |
|------|------|
| `/quote` | 현재 견적. 화면에서 수량 편집·삭제, 인쇄 / PDF 저장, 견적서 발행, 하단에 주문 버튼 |
| `/quotes/[number]` | 발행된 견적서 열람. 발행 시점에 고정된 읽기 전용 문서 — 수정하려면 상품을 다시 담아 새로 발행 |
| `/checkout` | 주문/결제. 배송정보·무통장입금·세금계산서 → 주문 완료 |

### 계정  `(plain)`

| 경로 | 설명 |
|------|------|
| `/login` | 로그인. 이메일을 공용 primary key로 사용 — 개인·사업자 구분 없이 이메일 + 비밀번호 |
| `/signup` | 회원가입 유형 선택 (개인 / 사업자) |
| `/signup/personal` | 개인회원 가입 — 이메일로 즉시 가입 |
| `/signup/business` | 사업자회원 가입 — 사업자등록증 확인 후 승인, 회원도매가 적용 |
| `/find-account` | 아이디·비밀번호 분실 시 고객센터 문의 안내 |

### 마이페이지  `(plain)`

| 경로 | 설명 |
|------|------|
| `/mypage` | 진입 시 결제내역으로 리다이렉트 |
| `/mypage/quotes` | 발행한 견적서 내역 (유효기간 표시) |
| `/mypage/orders` | 결제내역 (견적 단위) + 배송상태 뱃지 + 배송조회 타임라인 |
| `/mypage/tax-invoices` | 세금계산서 발행 내역 |
| `/mypage/addresses` | 배송지 등록·삭제·기본배송지 설정 |
| `/mypage/profile` | 상호·담당자·연락처·이메일·비밀번호 변경 |
| `/mypage/withdraw` | 회원 탈퇴 |

### 기타

| 경로 | 설명 |
|------|------|
| `/dev/seed` | **개발용.** `data/seed/*.json` 을 localStorage에 주입해 목업 사용자·주문·견적·배송지 테스트. 배포 시 제외 대상 |

## 진입 경로 (헤더/화면 내 링크)

- **헤더 우측:** 검색 · 견적서(`/quote`) · 마이페이지(`/mypage/orders`) · 로그인(`/login`) · **회원가입(`/signup`)**
- **카테고리 내비:** 각 카테고리 → 홈 앵커(`/#category-<slug>`)
- **좌측 견적 패널:** "견적서 보기" → `/quote`
- **`/quote`:** 수량 편집·삭제, 견적서 발행 → `/quotes/[number]`, "주문 / 결제하기" → `/checkout`
- **`/checkout` 완료:** "주문내역 보기" → `/mypage/orders`
